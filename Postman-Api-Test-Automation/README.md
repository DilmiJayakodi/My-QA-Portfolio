#  Automating API Test Flow with Postman Collections & Runner

##  Project Overview
This mini project demonstrates automated API workflow testing using Postman Collections and Collection Runner.

## Tools Used
- Postman
- Postman Collection Runner
- JSON
- GitHub

##  Project Structure
- Collection file (4+ API requests)
- Environment file (Base URL & dynamic variables)

## Features
- Complete API workflow execution
- Dynamic variable passing between requests
- Request chaining using environment variables
- Automated test validation using scripts

## Example Workflow
- Create user
- Get user
- Update user
- Delete user
